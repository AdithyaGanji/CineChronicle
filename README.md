# CineChronicle

CineChronicle is a fully responsive web application that lets you discover various movies and series.

## Features

- **Home Dashboard**: Browse curated selections including New Arrivals, Trending Movies, TV Series, and Trailers.
- **Search System**: A refined search experience with controls for media type selection and detailed results that show important attributes and credits.
- **Data Handling**: Graceful fallbacks for missing posters and API data.
- **Responsive Design**: Carefully crafted with vanilla CSS to look beautiful and function flawlessly across all devices.

## Tech Stack

- **Core**: HTML, TypeScript, React 
- **Styling**: Vanilla CSS
- **Build Tool**: Vite
